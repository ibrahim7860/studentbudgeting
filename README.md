
# What To Do?

An easy to use todo-list built using ReactJS for the frontend and Firebase for the backend. 

## Run Locally

Clone the project

```bash
  git clone https://github.com/ibrahim7860/todoapp.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
